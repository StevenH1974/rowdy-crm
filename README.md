# The Rowdy CRM

A custom-built Customer Relationship Management dashboard designed for The Rowdy Kernel (gourmet popcorn) and The Rowdy Pizza — two startup food businesses under the Rowdy Brands umbrella.

Built as a cost-effective alternative to paid platforms like HubSpot or Shopify, this CRM gives full control over business operations without monthly subscription fees.

## Features

- **Dashboard** — Live stat cards showing total orders, monthly revenue, inventory alerts, and new customers. Recent orders table updates in real time.
- **Orders** — Full order management with status filtering (New, Processing, Shipped) and color coded status badges.
- **Inventory** — Stock tracking for all products with automatic low stock and out of stock alerts based on quantity.
- **Pricing & Costing** — Product pricing table and a working profit margin calculator.
- **Reports** — Six report types including Sales, Orders, Inventory, Revenue by Month, Customer Acquisition, and Shipping Costs.
- **Documents** — Storage and management of supplier and distributor contracts.

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Database:** Supabase (PostgreSQL)
- **Version Control:** Git & GitHub
- **Local Development:** VS Code with Live Server

## Status

Actively in development. Currently connecting all sections to live Supabase data. Upcoming features include a Finances section, Customer profiles, print functionality, and Netlify deployment.
